# Instrucciones para los ejercicios de Desarrollo

Este repositorio contiene ejercicios prácticos para las áreas de Desarrollo Web, Mobile, Base de Datos y Python. Cada área tiene su propia carpeta y dentro encontrarás los archivos que debes completar o ejecutar según el enunciado.

## Estructura de carpetas

- `web-desarrollo/` — Ejercicios de JavaScript para navegador (HTML/JS)
- `mobile-desarrollo/` — Ejercicios de React Native (JSX)
- `db-desarrollo/` — Ejercicios de SQL para PostgreSQL
- `python-desarrollo/` — Ejercicios de Python

## ¿Qué debes hacer?

1. **Clona este repositorio en tu equipo.**
   - Si no tienes Git instalado, puedes descargar el ZIP desde GitHub y descomprimirlo.

2. **Ubica la carpeta del área correspondiente a tu evaluación.**

3. **Resuelve los ejercicios asignados:**
   - Abre cada archivo y sigue las instrucciones y comentarios dentro del código.
   - Si el ejercicio tiene secciones `// TODO` o comentarios similares, complétalas según lo solicitado.
   - Ejecuta el código para verificar que funciona correctamente.

4. **Herramientas recomendadas:**
   - **Web:** Puedes usar cualquier navegador moderno para abrir los archivos `.html` o ejecutar los scripts JS en la consola del navegador.
   - **Mobile:** Puedes usar [Snack Expo](https://snack.expo.dev/) para ejecutar archivos `.jsx` de React Native sin instalar nada.
   - **Base de Datos:** Puedes usar [DB Fiddle](https://www.db-fiddle.com/) o [SQL Fiddle](https://sqlfiddle.com/) para ejecutar los scripts SQL de PostgreSQL.
   - **Python:** Puedes usar [Replit](https://replit.com/) o [PythonAnywhere](https://www.pythonanywhere.com/) para ejecutar los scripts `.py` si no tienes Python instalado.

5. **Sube tus respuestas:**
   - Realiza un commit por cada ejercicio resuelto.
   - El mensaje de commit debe ser: `Ejercicio X: "Respuesta"` (por ejemplo: `Ejercicio 4: "Respuesta"`).
   - No incluyas los comandos de Git en este archivo, ya que también se evaluará tu conocimiento sobre el uso de Git.

## Notas importantes

- Lee cuidadosamente cada enunciado y sigue las instrucciones específicas de cada archivo.
- Si tienes dudas sobre cómo ejecutar algún archivo, consulta las herramientas web sugeridas.
- Asegúrate de que tu solución funcione antes de subirla.
- No modifiques archivos fuera de tu área asignada.

¡Éxito en tu evaluación!
